# Interstellar Absorption and the Lyman-Alpha Forest

An interactive educational module for teaching stellar absorption lines and the Lyman-alpha forest.

**Live Demo**: [https://tingyuansen.github.io/Lyman_Alpha_Module/](https://tingyuansen.github.io/Lyman_Alpha_Module/)

## Features

- **Stellar Line Formation Module**: Explore how different elements create absorption lines in stellar spectra
- **Lyman-Alpha Forest Module**: Understand how quasar light is absorbed by intergalactic hydrogen
- Interactive sliders to adjust parameters like column density, Doppler width, and redshift
- Video tutorial available
- Both public and student (with mathematical details) text versions
